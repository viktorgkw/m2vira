## Getting Started

- Clone the project

```bash
    git clone https://github.com/viktorgkw/m2vira.git
```

- Hop into the project folder

```bash
    cd m2vira
```

- Install the dependencies

```bash
    npm i
```

- Start the app in development mode

```bash
    npm run dev
```
